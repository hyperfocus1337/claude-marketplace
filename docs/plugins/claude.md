# Installed Claude Code plugins
Currently installed Claude Code plugins.

## Claude Code

### Official
Official Claude Code [plugins](https://github.com/anthropics/claude-code/tree/main/plugins/code-review)
- [feature-dev](https://github.com/anthropics/claude-code/tree/main/plugins/feature-dev)
- [commit-commands](https://github.com/anthropics/claude-code/tree/main/plugins/commit-commands)
- [code-review](https://github.com/anthropics/claude-code/tree/main/plugins/code-review)

```
claude plugin install code-review@claude-code-plugins
claude plugin install commit-commands@claude-code-plugins
claude plugin install feature-dev@claude-code-plugins
```

### Context7 plugin
Context 7 [docs](https://github.com/upstash/context7/tree/master/plugins/claude/context7)
```
claude plugin marketplace add upstash/context7
claude plugin install context7-plugin@context7-marketplace
```

### Whobson agents
Claude Code [plugins collection](https://github.com/wshobson/agents)
```
claude plugin marketplace add wshobson/agents
claude plugin install code-refactoring@claude-code-workflows
```
